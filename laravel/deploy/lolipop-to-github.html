ロリポップマネージドクラウドとGitHubのSSH通信設定

ここまででローカルPCからマネージドクラウド上の仮想マシンを操作できるようになった．

続いて，マネージドクラウド上にGitHubのソースコードをデプロイするため，両者の間で通信できるように設定を行う．

ssh-keyの発行

🔗 Lolipopでの操作

ssh-keyを作成し，マネクラとGitHubの間で通信できるようにする．

$ cd .ssh
$ ssh-keygen
